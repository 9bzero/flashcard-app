# Flashcard App

Spaced repetition flashcard study app — create decks, study with confidence ratings, and track your progress.

## How it works

Uses a simplified **SM-2** spaced repetition algorithm. Cards you mark as hard appear sooner; cards you know well are shown less frequently. This maximizes retention with minimal study time.

## Features

- Create and manage multiple decks
- Add, edit, and delete cards
- Study mode with flip animation
- Confidence rating (Again / Hard / Good / Easy)
- Progress tracking: mastered, learning, new
- Study session stats and streaks
- Import / export decks as JSON
