# flashcard-app

![TypeScript](https://img.shields.io/badge/TypeScript-007ACC?style=flat-square&logo=typescript&logoColor=white) ![React](https://img.shields.io/badge/React-20232A?style=flat-square&logo=react&logoColor=61DAFB) ![Vite](https://img.shields.io/badge/Vite-646CFF?style=flat-square&logo=vite&logoColor=white)

> Spaced repetition flashcard app with 3 built-in study decks

## Features
- 🃏 3 decks: TypeScript, React, CS Fundamentals
- ✓ Mark cards as known to track progress
- 🔄 Flip animation on click to reveal answer
- 📊 Progress bar and card counter
- 🎯 15 real interview-quality Q&A cards

## Tech Stack
`React 18` · `TypeScript` · `Vite`

## Quick Start
```bash
npm install && npm run dev
```
