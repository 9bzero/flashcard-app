# flashcard-app

Study with flashcards and spaced repetition.

Cards you get wrong come back sooner. Cards you know well come back later. Based on a simplified SM-2 algorithm.

## Features

- Create decks and add front/back cards
- Study mode with flip animation
- Rate each card: Easy / Hard / Again
- Spaced repetition scheduler
- Progress stats per deck

## Run

```bash
npm install && npm run dev
```
